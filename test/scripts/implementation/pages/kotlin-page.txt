<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="infobox" style="width: 2in"><div class="floatleft"><a href="/wiki/Category:Solutions_by_Programming_Language" title="Category:Solutions by Programming Language"><img alt="Language" src="/mw/images/thumb/2/27/Rcode-button-language-crushed.png/64px-Rcode-button-language-crushed.png" width="64" height="64" srcset="/mw/images/thumb/2/27/Rcode-button-language-crushed.png/96px-Rcode-button-language-crushed.png 1.5x, /mw/images/thumb/2/27/Rcode-button-language-crushed.png/128px-Rcode-button-language-crushed.png 2x"/></a></div><b>Kotlin</b><br/> This <b>programming language</b> may be used to instruct a computer to perform a task.

Listed below are all of the tasks on Rosetta Code which have been solved using Kotlin.</div><div class="infobox" style="width: 2in"><big><b>Your Help Needed</b></big><br/>If you know <b>Kotlin</b>, please write code for some of the <a href="/wiki/Reports:Tasks_not_implemented_in_Kotlin" title="Reports:Tasks not implemented in Kotlin">tasks not implemented in <b>Kotlin</b></a>.</div>Kotlin is a statically-typed programming language that runs on the <a href="/wiki/JVM" title="JVM" class="mw-redirect">JVM</a> and can also be compiled to JavaScript. It is being developed by JetBrains with assistance from open source contributors and is intended for industrial use.
<p><br/>
In addition, a separate team at JetBrains is developing a version of Kotlin which compiles (via LLVM) directly to machine code for various platforms. The first Technology Preview for this version was rolled out in April 2017.
</p><p>Kotlin has become a popular language for developing Android applications and, in May 2017, Google announced official support for it alongside Java and C++.
</p>
